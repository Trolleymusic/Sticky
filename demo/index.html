Sticky

jQuery plugin for making things stick

Elements get stuck as you scroll. A regular call to Sticky will keep the element fixed as long as your scrollTop is greater than its initial offset on the page.

$(element).Sticky();

Scroll down further for some more examples.

They can get unstuck when they've traveled their own height

$(element).Sticky({ bottom : true });

Or they can get unstuck when they reach the bottom of their container element


$(element).Sticky({ bottom : true,
                    reference : $(container) });

Usually they can sit flush against the top of the window

But you can also give them a buffer


$(element).Sticky({ bottom : true,
                    reference : $(container),
                    buffer : 50 });

Sticky's not CSS independent at the moment. Sticky will add/remove whichever class you specify, and also fix the element's bottom positioning if the bottom flag is used but you also need to include the following css:

.stuck { position : fixed; top : 0; }

stuck is the default keyword, but you can use a different one by including keyword : "whatever" in the settings object when you call Sticky.
